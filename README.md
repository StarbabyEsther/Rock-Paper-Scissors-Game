# Rock Paper Scissors 

A Rock Paper Scissors game built in Java.

## About
The user picks rock, paper or scissors and 
the computer picks randomly. 
The program then decides who wins!

## What I learned
- How to use .equals() to compare Strings 
  instead of == which is only for numbers
- How to combine two conditions in an if 
  statement using &&
- How to map random numbers to real choices
  (0 = rock, 1 = paper, 2 = scissors)

## Built with
- Java
